Main --> Service : масив
Service --> JavaCalc : average(values)
Service --> NativeCalc : average(values)

NativeCalc ..> Header : javac -h
Header --> Cpp : JNI-сигнатура
Cpp --> Lib : clang++
NativeCalc --> Path : System.loadLibrary()
Path --> Lib : пошук бібліотеки
Lib --> NativeCalc : результат

note right of NativeCalc
Java-клас оголошує native-метод
і завантажує бібліотеку.
end note

note bottom of Lib
Windows: nativeaverage.dll
Linux CI: libnativeaverage.so
end note
